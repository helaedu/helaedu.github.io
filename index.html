Auto Vote Clicker — Privacy Policy

What this extension does

Opens a specified voting page in a background tab at a scheduled time (or when you click “Vote”) and clicks the Vote/Change Vote button once. The tab auto-closes.

Data the extension processes

We read minimal content from the declared voting domains to detect:

  • Button state: “Vote”, “Change Vote”, or “Voted”.
  • Visible login email string (if the site shows it) to confirm you are logged in.

We record a log entry containing timestamp(s), page URL, entry ID, result (success / already_voted / timeout / not_logged_in / error), and the visible email if present.

Where data is sent

Log entries are posted to a Google Apps Script URL owned by the developer for the sole purpose of keeping run history. Data is not sold or shared with third parties.

Permissions

Tabs (open background tab), Alarms (daily schedule), Storage (local settings/logs), Downloads (export logs), Host permissions (voting site + the Apps Script endpoint).

User control

You can delete the extension at any time (chrome://extensions) and remove your local logs via the browser.
